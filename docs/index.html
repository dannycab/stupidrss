StupidRSS

A fast, minimalistic RSS reader built with FastAPI and Tailwind CSS. No bloat, just pure RSS reading experience with a modern interface.

Tech Stack

Modern technologies for maximum performance

Backend

  • • FastAPI - Modern async Python web framework
  • • SQLAlchemy - Python SQL toolkit and ORM
  • • SQLite - Lightweight database
  • • feedparser - RSS/Atom feed parsing
  • • httpx - Async HTTP client

Frontend

  • • Jinja2 - Template engine
  • • Tailwind CSS - Utility-first CSS framework
  • • Vanilla JavaScript - No heavy frameworks
  • • Dark Mode - System preference aware
  • • Responsive Design - Mobile-first approach

Quick Start

Get StupidRSS running in minutes

# Clone the repository
git clone https://github.com/dannycab/stupidrss.git
cd stupidrss

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py

# Open your browser to the RSS reader
open http://localhost:8000/app

# View interactive API docs 
open http://localhost:8000/docs
